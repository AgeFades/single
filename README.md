## 简介

- 基于 SpringBoot 的极简单体应用基础架构
- 只依赖于 MySQL8 + Redis

## 目的

- AgeFades 整理优化公共、可复用的高质量代码

## 功能

- 基于 JWT Filter 做 `认证、鉴权` 的 RBAC 权限模型 
- 基于 Log4j2 的日志体系
- 操作日志、HttpUtil、业务断言工具类、全局异常捕获、请求串联日志traceId
- ...
